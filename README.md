# sharecar-rails

## Setup

Please install docker first.

```shell-session
$ ./launch_db.sh
$ bundle install --path vendor/bin
$ bundle exec rake db:setup
$ bundle exec rails s
```

Go to [http://localhost:3000](http://localhost:3000).

This webapp is for smartphone's web browser.

## Restore from heroku

```shell-session
./pull_db.sh
```
